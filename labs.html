/soft

Разные программки, написанные для себя:

  • HSE Timetable Checker
    Программа-менеджер для проверки изменений на странице расписания ГУ-ВШЭ (ПФ).
  • BF2 AIX loader
    Программа запускает BF2 с модом AIX2. C возможностью настройки кол-ва ботов или же выбор сервера к которому коннектится.
  • FastAlarms
    Программа-будильник, с неограниченным числом будильников, возможностью включать/отключать их и приятным интерфейсом. Написана для напоминания о включении ТВ и просмотра нужных передач.
  • Система Тестирования (с исходниками)
    Система для тестирования людей (учеников). Является хорошим примером работы плагинов на основе dll библиотек.

/la2soft

Написанный мной софт для Lineage 2 (с исходниками для Delphi 7):

  • Clock in Game
    Часы в игре. Расположение, цвет регулируются. Показывает сколько времени окно работает.
  • L2Hider
    Прячет окна с панели задач.
  • L2Presser
    Поочередное нажатие разных кнопок одной кнопкой.
  • La2AutoFishing
    Авторыбалка для интерлюд.
  • La2AutoManor
    Удобный кликер (автосдача манора).
  • La2Control
    Управление фоновыми окнами, проверка онлайна серверов, запуск окон, проверка пикселей на экране.
  • La2 DL Helper 2
    Помошнег ДЛа, для старого рейда.
  • L2WindowName
    Присваивает окнам введенные имена, на панели задач.
